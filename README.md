# happyGo 电商后台管理系统

### 前言

该项目是我的第一个 vue 项目，跟着 coderwhy 做的

### 技术栈

```
vue2
vuex
vue-router
less
axios
better-scroll
fastclick
mockjs
vue-lazyload
```

### 安装项目依赖

```js
npm install
```

### 运行项目

```js
npm run server

```
